JSON, acrónimo de JavaScript Object Notation, es un formato de texto ligero para el intercambio de datos. ## JSON
AJAX es el acrónimo de Asynchronous Javascript and XML, es decir: Javascript y XML Asincrono ## AJAX
Una API es un conjunto de funciones y procedimientos que cumplen una o muchas funciones con el fin de ser utilizadas por otro software. ## API
Windows Communication Foundation (WCF) es un marco de trabajo para la creación de aplicaciones orientadas a servicios. ## WCF
